AI image generation

Install guide

Use this page when the extension shows Not ready or when you need to reload the unpacked folder.

1. Download

Get the AI image generation folder from your Drive and unzip it locally.

2. Load unpacked

Open chrome://extensions, enable Developer mode, then choose Load unpacked.

3. Select the folder

Pick the unzipped AI image generation directory that contains manifest.json.

4. Verify

Open the side panel again. The status should switch from Not ready to Queue ready after import.

  1. Keep the extension folder unchanged after install.
  2. If the panel says Offline, reload the extension from chrome://extensions.
  3. Use Load from SRT2Prompt after generating a content pack.
